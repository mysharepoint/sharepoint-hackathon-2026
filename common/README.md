# Common

This folder contains the shared organisation context for the demo.

It is the identity layer that keeps all outputs aligned to the same fictional
organisation.

## What Is In Here

- `council-profile-bwec.md`
  Core profile of the fictional British Wind Energy Council
- `press-profile-bwec.md`
  Press-facing boilerplate, media contact details, and release footer material

## Why It Matters

These files give the agent a stable organisational identity.

They define:

- who BWEC is
- what the organisation stands for
- how it presents itself externally
- which boilerplate and contact details belong in media outputs

Use this folder as the common context layer beneath all formats.
