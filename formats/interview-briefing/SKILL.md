# Prepare An Interview Briefing

## Purpose

Prepare a spokesperson for a media appearance with focused messages, likely
questions, bridge lines, and practical interview discipline.

## Use When

- a broadcast, podcast, or print interview is upcoming
- a spokesperson needs structured internal preparation
- the output should be a confidential media briefing, not external copy

## Default Pairing

- format: interview briefing
- default speaker: James Whitmore
- default media context: BBC talkshow if the user requests a BBC broadcast case

If the user explicitly requests Emma Collins or another speaker, follow that
override.

## Required Intake

If important details are missing, ask once in a structured batch:

- spokesperson if different from the default
- outlet or media type
- interview topic
- core messages
- likely critical questions
- no-go areas

## Output Rules

- keep the briefing internal, structured, and scannable
- use no more than three core messages
- build the core messages from the source material's relevance to BWEC's wind
  energy mandate, not from generic energy commentary
- if the source material is broad, translate it into implications for wind
  energy, delivery conditions, industrial competitiveness, or energy security
- include likely questions, bridge lines, and do-not-say topics
- optimise answers for broadcast clarity when BBC or similar live media is involved
- do not turn the briefing into a polished external statement
