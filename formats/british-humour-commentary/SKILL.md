# Write A British-Humour Commentary

## Purpose

Turn source material into a short, memorable commentary that combines dry
British wit with a serious practical point.

This format is opinionated by design. A generic witty commentary is a failed
result.

## Use When

- a public-facing commentary should sound sharper than standard corporate copy
- a short video, podcast, or social-ready monologue is needed
- the goal is to expose delay, contradiction, or institutional absurdity
  without losing seriousness

## Default Pairing

- format: British-humour commentary
- default speaker: Sir Alistair Pembroke
- default media context: British Humour
- default intensity: Strong

This format is not the default for crisis statements or formal press releases.

## Required Intake

If important details are missing, ask once in a structured batch:

- source issue or trigger
- desired channel if relevant
- key practical argument
- length expectation
- humour intensity if the user explicitly requests `Light` or `Strong`
- any sensitive themes that must be handled carefully

## Output Rules

- lead with a sharp observation or contradiction
- use dry wit, light irony, elegant contrast, and mild sardonic pressure
- keep the humour in service of clarity
- make the serious point clearly relevant to wind energy, BWEC's sector
  mission, or the delivery conditions for the wind sector
- if the source material is broad, turn the contradiction into a practical
  point about wind energy, infrastructure, delivery, or industrial common
  sense
- include one line that politely exposes absurdity or delay
- include one deadpan turn or faintly absurd comparison that makes the
  institutional logic feel ridiculous without becoming silly
- land on one serious, practical point
- prefer amused exasperation over neutral diplomacy
- treat the format choice itself as enough to activate the full Pembroke voice
- avoid parody, slapstick, and celebrity imitation

If the user asks for `British Humour Light`, use a more restrained, polished,
essayistic version of the format.

If the user asks for `British Humour Strong`, or if no intensity is specified,
use the full absurdist Pembroke register.
