# Draft A Press Release

## Purpose

Create a press-ready BWEC release that follows a recognisable journalistic
structure and can serve as a credible basis for media distribution.

## Use When

- a public comment or announcement needs a formal press release
- uploaded or pasted source material should be turned into external press copy
- the user wants a media-ready BWEC release rather than an internal briefing

## Default Pairing

- format: press release
- default speaker: Emma Collins
- default organisation context: BWEC boilerplate and BWEC press contact

If the user explicitly requests James Whitmore or another named speaker, follow
that override.

## Required Intake

If important details are missing, ask once in a structured batch:

- trigger or announcement
- core message
- source-backed facts
- quote owner if different from the default
- embargo status
- press contact override if different from BWEC

## Output Rules

- follow an inverted-pyramid structure
- keep the tone factual, media-ready, and quotable
- make the relevance to wind energy and BWEC's sector mandate explicit where
  the source material supports it
- if the source material is broader than wind energy, frame the release around
  the practical consequences for wind energy deployment, supply chains, energy
  security, project delivery, or investment confidence
- use BWEC boilerplate and press contact by default
- mark quotes as `APPROVAL PENDING` unless explicitly cleared
- do not invent titles, organisations, or contact details
