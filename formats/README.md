# Formats

This folder contains the three core output formats as internal skill
definitions.

## Current Format Skills

- `press-release/`
  Public media statement for the organisation
- `interview-briefing/`
  Internal spokesperson prep for broadcast or press interviews
- `british-humour-commentary/`
  Character-driven public commentary for social, video, or podcast use

## Why It Matters

These skill folders define what the agent is actually supposed to produce.

They are the structural source of truth for:

- output purpose
- expected structure
- default speaker pairing
- rhetorical intent

In Copilot Studio, these are mirrored into the flat upload pack as simplified
format files.
