# Demo Assets

## Purpose

This folder stores demo artefacts created from live test runs or presentation
production steps.

These files are not part of the knowledge pack and should not be uploaded as
agent knowledge in Copilot Studio.

## Current Use

- `british-humour-output.md` for the saved humour-style output from the live
  prompt
- `talking-head-video-notes.md` for the linked video asset and production notes
- `speaker-portrait-prompts.md` for consistent image-generation prompts for the
  three BWEC speaker personas

## Recommended Conventions

- Keep generated demo outputs here, not in `copilot-studio/`
- Use descriptive file names tied to the actual demo path
- Store videos next to their notes when they are ready
- Treat this folder as presentation material, not retrieval knowledge
