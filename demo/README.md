# Demo

This folder contains the live-demo material that sits around the actual skill
package.

It helps turn the skill pack into something that can be shown, rehearsed, and
explained quickly.

## What Is In Here

- `demo-prompts.md`
  Longer operator prompts for rehearsed demo runs
- `copilot-studio-build-runbook.md`
  Build and setup narrative for the MVP version
- `project-readiness.md`
  Short status and readiness notes
- `assets/`
  Reusable public-facing demo assets such as sample outputs and prompt helpers

## Why It Matters

The `demo/` folder is where the package becomes presentable.

It helps explain:

- what to show
- in which order to show it
- which prompts to run
- which supporting assets already exist

This folder is useful for rehearsals, recordings, and live walkthroughs.
