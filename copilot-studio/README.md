# Copilot Studio

This folder contains the operational material needed to build and run the demo
agent in Microsoft Copilot Studio.

## What Is In Here

- `upload-pack/`
  The flat file set to upload as embedded knowledge
- `agent-instructions.txt`
  The main instruction block for the agent
- `agent-description.txt`
  Short description for the Copilot Studio UI
- `welcome-message.txt`
  Startup message shown to users
- `suggested-prompts.md`
  Visible prompt chips for the demo
- `runtime-inputs/`
  Example source material used during testing and demos
- build, publish, and update checklists

## Why It Matters

This is the practical handoff layer between the skill package and the actual
agent setup in Copilot Studio.

If someone wants to rebuild the demo quickly, this folder is the shortest path.
