# Docs

This folder contains the public-facing guidance that explains how to build a
skill-based communication package like the one used in this repository.

It is the best entry point for understanding the method behind the project and
the thinking behind the skill structure.

## What Is In Here

- `how-to-build-a-skill-pack.md`
  End-to-end guide for defining a use case, designing skill layers, and
  turning those decisions into reusable Markdown files

## Why It Matters

The most important work in this project happened before the agent was built in
Copilot Studio.

This documentation focuses on that preparation work:

- defining the use case
- structuring the skill layers
- gathering wording, speaker, media, and format logic
- using AI tools as drafting support

That same preparation will also matter for future SharePoint-native skill
experiences.
